## ✨ Fitur

| Fitur | Deskripsi |
|-------|-----------|
| 🔍 Pencarian | Cari film & series dari database TMDB |
| 🎬 Film Populer | Update otomatis film terbaru & trending |
| 📺 Series Populer | Serial TV populer subtitle Indonesia |
| ⭐ Rating Tertinggi | Film dengan rating terbaik |
| ▶️ Streaming | Multi-server player (VidSrc, 2Embed) |
| 📱 Responsive | Tampilan optimal di mobile & desktop |
| 🌙 Dark Mode | Tema gelap yang nyaman di mata |
| 🎥 Trailer | Tonton trailer langsung dari YouTube |

## 🚀 Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JavaScript
- **API** — [The Movie Database (TMDB)](https://www.themoviedb.org/)
- **Player** — VidSrc / 2Embed (third-party embed)
- **Hosting** — GitHub Pages

## 📂 Struktur

```
nonton-film/
├── index.html      # Halaman utama
├── style.css       # Styling (dark theme, responsive)
├── app.js          # Logika aplikasi & API calls
└── README.md       # Dokumentasi
```

## 📝 Catatan

> Semua konten film & series disediakan oleh pihak ketiga. Kami tidak menyimpan file video di server kami. Gunakan untuk referensi dan hiburan saja.
